# Terminal GPT

This tool lets you use ChatGPT 3.5 in Terminal without any API KEYS. It communicates with the Backend of [Bai chatbot](https://chatbot.theb.ai).

Usage
```
./tgpt "Explain quantum computing in simple terms"
```
